Introduction

JavaScript is a versatile programming language primarily used to create dynamic and interactive content on websites. It is the backbone of web development, alongside HTML and CSS, and continues to evolve with new features and improvements every year. This documentation covers the basics of JavaScript syntax and introduces some new features available in 2025.

What you should already know

Before diving into JavaScript, it’s helpful to have a basic understanding of:

  • HTML: The structure of web pages, including elements and attributes.
  • CSS: Styling web pages using classes, IDs, and selectors.
  • Programming Fundamentals: General concepts like variables, loops, and conditionals (even if from another language).

Having this foundational knowledge will make learning JavaScript much easier and more intuitive.

JavaScript vs Java

Despite their similar names, JavaScript and Java are fundamentally different:

  • Purpose:
    • JavaScript: A scripting language designed to add interactivity to web pages. It runs in browsers and is primarily used for front-end and back-end web development.
    • Java: A general-purpose programming language used for building applications, including web, mobile, and desktop software.
  • Execution Environment:
    • JavaScript: Runs in web browsers and uses a Just-In-Time (JIT) compiler.
    • Java: Runs in the Java Virtual Machine (JVM) and is compiled into bytecode before execution.
  • Typing:
    • JavaScript: Dynamically typed (variable types are determined at runtime).
    • Java: Statically typed (variable types are declared explicitly).
  • Syntax:
    • JavaScript: Lightweight and flexible syntax, optimized for the web.
    • Java: Heavier, object-oriented syntax with stricter rules.
Basic Programming Concepts

Before diving into JavaScript specifics, let’s review a few key programming concepts that form the foundation of any programming language:

Variables

Definition: Variables are containers for storing data values. They allow programs to manipulate and keep track of information.

Example: let age = 25; // Variable to store age const name = "Alice"; // Constant variable for name

Data Types

Definition: Data types classify the type of value a variable can hold, such as numbers, strings, and booleans.

Examples: let number = 42; // Number let text = "Hello!"; // String let isOnline = true; // Boolean

Functions

Definition: Functions are blocks of reusable code designed to perform specific tasks.

Example: function add(a, b) { return a + b; // Adds two numbers and returns the result } console.log(add(2, 3)); // Output: 5

Conditionals

Definition: Conditionals allow a program to make decisions and execute different code paths based on certain conditions.

Example: if (age >= 18) { console.log("You are an adult."); } else { console.log("You are a minor."); }

Loops

Definition: Loops allow repetitive execution of code as long as a specified condition is true.

Examples: // For loop for (let i = 0; i < 5; i++) { console.log(i); // Prints 0, 1, 2, 3, 4 } // While loop let count = 0; while (count < 3) { console.log(count); // Prints 0, 1, 2 count++; }

Basics of JavaScript Syntax

Adding JavaScript to a Web Page

You can add JavaScript to an HTML file using the script tag:

            <html lang="en">
            <head>
                <title>JavaScript Example</title>
            </head>
            <body>
                <h2>Welcome!</h2>
                <script>
                    console.log("Hello World!");
                </script>
            </body>
            </html>
Variables

Variables are used to store data. You can declare variables in three ways:

let message = "Hello!"; // Block-scoped and can be updated const pi = 3.14; // Block-scoped and cannot be updated var name = "John"; // Function-scoped (older syntax)
Data Types

JavaScript supports various data types:

  • Number: let age = 30;
  • String: let greeting = "Hi!";
  • Boolean: let isOnline = true;
  • Array: let fruits = ["apple", "banana", "cherry"];
  • Object: let user = { name: "Alice", age: 25 };

Printing to the Console

Use console.log() to display messages in the browser's console:

console.log("JavaScript is fun!");
Basic Features
Functions

Functions are reusable blocks of code:

function greet(name) { return `Hello, ${name}!`; } console.log(greet("Alice"));

Or use arrow functions for a shorter syntax:

const add = (a, b) => a + b; console.log(add(2, 3)); // Output: 5

Conditionals

Conditionals let you perform different actions based on conditions:

let age = 18; if (age >= 18) { console.log("You are an adult."); } else { console.log("You are a minor."); }
Loops

Loops allow you to repeat actions:

for (let i = 0; i < 5; i++) { console.log(i); } const fruits = ["apple", "banana", "cherry"]; fruits.forEach(fruit => console.log(fruit));
New JavaScript Features (2025)
pipeline operator (|>)

Makes chaining functions cleaner:

const double = x => x * 2; const increment = x => x + 1; const result = 5 |> double |> increment; console.log(result); // Output: 11

do expressions

Allows evaluating multiple statements:

let result = do { if (age > 18) { "Adult"; } else { "Minor"; } }; console.log(result);

Records and Tuples

Immutable data structures:

const record = #{ name: "Alice", age: 25 }; const tuple = #[1, 2, 3]; console.log(record.name); // Output: Alice console.log(tuple[0]); // Output: 1
